## Cara menggunakan di perangkat local anda

Clone project

```bash
  git clone https://github.com/dimas292/portofolio.git
```

Pindah ke Direktori yang anda buat

```bash
  cd my-project
```

Install yang diperlukan 

```bash
  npm install
```

start untuk menjalankan kode

```bash
  npm run dev
```
